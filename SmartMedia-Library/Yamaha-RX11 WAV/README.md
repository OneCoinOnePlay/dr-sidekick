# Yamaha RX11 - SP-303 Pack

Yamaha RX11 drum machine samples for Boss DR. Sample SP-303

## Contents

- **Samples:** 8 drum sounds
- **Bank:** C (Pads 1-8)

## Installation

1. Load this pack in SP-303 Wizard
2. Or manually:
   - Copy `banks/C/wav/*.wav` to SmartMedia as `SMPL0001.WAV`, etc.
   - Run SP-303 import (BANK C → REC)
